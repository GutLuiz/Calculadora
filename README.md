# CALCULADORA 
-> Projeto C# Academy.
  - Conclua o seguinte tutorial (partes 1 e 2): Criar um aplicativo de calculadora (Microsoft Docs) . Você encontrará um link para a segunda parte na parte inferior da página.
  - Crie uma funcionaldiade que conte quantas vezes a calculadora foi usada
  - Armazene uma lista com os calculos mais recentes e permita que os usuarios excluam essa lista
  - Permita que os usuarios utilizem os resultados da lista acima para realizar novos calculos
